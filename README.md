  <h1>üéÆ Echo Caves</h1>
  
  <p><strong>Echo Caves</strong> is a 2D sound-based adventure game developed as a course project for 
  <strong>CSE 102: Structured Programming Language</strong> during Level 1, Term 1 at the 
  <strong>Department of Computer Science and Engineering, BUET</strong>.</p>

  <p>The game is built using the <strong>iGraphics</strong> graphics library in C. Players take on the role of an explorer trapped in a dark cave system, where vision is replaced by echolocation. Navigation depends on emitting sonar-like pulses to reveal obstacles and pathways momentarily.</p>

  <h2>üîß Features</h2>
  <ul>
    <li>Exploration using echolocation-based mechanics</li>
    <li>2D sprite animations for character movement</li>
    <li>Sound feedback for cave environment awareness</li>
    <li>Menu, gameplay, game over, and leaderboard screens</li>
    <li>Leaderboard with file I/O for score storage</li>
    <li>Modular C code using functions and logical separation</li>
  </ul>
